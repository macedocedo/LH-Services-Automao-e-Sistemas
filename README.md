# ⚡ LH Services – Automação e Sistemas

Site institucional da **LH Services**, empresa especializada em instalações elétricas, iluminação de alto padrão, automação residencial e sistemas de segurança, atendendo Campinas e região.

---

## 🖥️ Sobre o Projeto

Landing page desenvolvida em **HTML, CSS e JavaScript puro**, sem frameworks ou dependências externas. O foco foi criar uma experiência rápida, responsiva e visualmente impactante para apresentar os serviços da empresa e converter visitantes em clientes via WhatsApp.

---

## ✨ Funcionalidades

- Accordion de serviços — exibe 1 card por vez com animação suave
- Contador animado de estatísticas ao rolar a página
- Botão de orçamento direto no WhatsApp em cada serviço
- Menu responsivo com hamburger para mobile
- Botão de voltar ao topo
- Animações de entrada com Intersection Observer
- Design dark com identidade visual em preto e amarelo

---

## 🛠️ Tecnologias

- HTML5
- CSS3 (variáveis CSS, animações, grid, flexbox)
- JavaScript vanilla (sem jQuery ou frameworks)
